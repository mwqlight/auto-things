# 比佳佳商城
这是一个名叫比佳佳的商城，它的主要功能和京东商城，淘宝，抖音商城，但是却根本不一样，首先她没有登陆，无需登陆即可浏览商品，购买商品。但是她购买商品的逻辑是很不一样的，她不负责任何商品，她是各个商城的商品的搬运工，也就是说的核心功能，就是比价，她会利用用户检索的商品关键信息，去各大商城去爬取商品的核心信息，比如价格，评价，品牌，销量，参数等，然后形成一个对比图表，然后利用AI大模型给出建议。
## 核心功能
1. 关键词搜索爬取各个商城的商品信息，然后形成对比图表，这个图表你可以自由发挥。
2. 需要利用spring-ai-alibaba来集成大模型，要启用的AI建议的话，就提示用户配置大模型的参数，这里就涉及到reload的问题，因为用户配置的参数，是需要在运行时才生效的，所以这里需要在运行时，根据用户配置的参数，来动态加载大模型。
3. 技术栈是springboot+vue3+element-plus，前端利用vue3+element-plus来实现，后端利用springboot来实现，数据库利用mysql来实现。数据库信息如下：
    - 数据库名称：auto_things
    - 数据库用户名：root
    - 数据库密码：root
    - 数据库端口：3306
    - 数据库主机：localhost
redis信息如下：
    - redis主机：localhost
    - redis端口：6379
    - redis密码：
    - redis数据库索引：0
4. 需要集成开源的爬虫工具，然后将关键信息爬取下来，存储到数据库中。就会形成用户的搜索记录，然后根据用户的搜索记录，来推荐商品。
5. 没有用户登陆，直接就可以使用，刚开始就可以搜索商品，然后对比价格，购买商品。其他的功能就类似于京东商城，可以自由发挥。
6. 开发完毕后，启动前后端，然后没问题，推送到远程。
